# Aplicaciones de chat con Streamlit

Vamos a construir una aplicación sencilla que nos permita conversar con un LLM usando una API-key gratuita de [Groq](https://groq.ai/).

```python
import streamlit as st
import requests
import os
from dotenv import load_dotenv
```

### 1. Cargar las variables de entorno desde el archivo .env
```python
load_dotenv()
API_KEY = os.getenv("API_KEY_GROQ")
```

### 2. Configurar la página (opcional, pero mejora la estética)
```python
st.set_page_config(page_title="Chat con Groq", page_icon="🤖")
```

### 3. Inicializar el historial del chat en session_state
```python
if "messages" not in st.session_state:
    st.session_state.messages = []
```

### 4. Comprobar que la API key existe
```python
if not API_KEY:
    st.error("⚠️ No se encontró la variable API_KEY_GROK en el archivo .env")
    st.stop()
```

### 5. Interfaz del chat
```python
st.title("🤖 Chatbot con Groq")
```

### Mostrar los mensajes anteriores del historial
```python
for message in st.session_state.messages:
    with st.chat_message(message["role"]):
        st.markdown(message["content"])
```

### 6. Función para llamar a la API de Groq
```python
def obtener_respuesta_groq(historial):
    url = "https://api.groq.com/openai/v1/chat/completions"
    headers = {
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json"
    }
    
    # Formato que espera Groq (igual que OpenAI)
    payload = {
        "model": "llama-3.3-70b-versatile",
        "messages": historial,
        "temperature": 0.7
    }
    
    response = requests.post(url, json=payload, headers=headers)
    
    if response.status_code == 200:
        data = response.json()
        return data["choices"][0]["message"]["content"]
    else:
        return f"❌ Error al conectar con Groq: {response.status_code} - {response.text}"
```

### 7. Input del usuario y lógica del chat
Los mensajes se estructuran como diccionarios con dos claves:
- **`role`**: indica el emisor, puede ser `user` o "assistant".
- **`content`**: contiene el texto del mensaje.
El historial completo (`st.session_state.messages`) es una lista de estos diccionarios que se enví­a directamente a la API de Groq en el campo `messages` del payload, siguiendo el formato estándar de OpenAI.

```python
user_input = st.chat_input("Escribe tu mensaje aquí...")

if user_input:
    # Añadir el mensaje del usuario al historial y mostrarlo
    st.session_state.messages.append({"role": "user", "content": user_input})
    with st.chat_message("user"):
        st.markdown(user_input)
    
    # Generar y mostrar la respuesta del asistente
    with st.chat_message("assistant"):
        with st.spinner("Pensando..."):
            respuesta = obtener_respuesta_groq(st.session_state.messages)
            st.markdown(respuesta)
        
    # Añadir la respuesta al historial
    st.session_state.messages.append({"role": "assistant", "content": respuesta})
```
