# 07. Inclusión de tablas

- Mostrar datos tabulares con `st.dataframe()`, `st.table()` y `st.data_editor()`

    ```python
    # En el archivo dataframes_tabla.py
    df = pd.DataFrame([
        {"command": "st.selectbox", "rating": 4, "is_widget": True},
        {"command": "st.balloons", "rating": 5, "is_widget": False},
        {"command": "st.time_input", "rating": 3, "is_widget": True}])
    
    # Vista estática del dataframe.
    st.write("DataFrame visto con st.table")
    st.table(df)
    st.divider()
    
    # Vista formateada del dataframe. Puede descargarse.
    st.write("DataFrame visto con st.dataframe")
    st.dataframe(df)
    st.divider()
    
    # Con data_editor se puede editar el contenido y extraer el valor modificado.
    st.write("DataFrame visto y editable con st.data_editor")
    edited_df = st.data_editor(df, num_rows="dynamic")
    favorite_command = edited_df.loc[edited_df["rating"].idxmax()]["command"]
    st.markdown(f"Your favorite command is **{favorite_command}**")
    ```

- Configuración de columna: `st.column_config`

    Es una característica útil para configurar cómo se muestran y se utilizan los datos. Está diseñada para usarse con la opción `column_config` tanto en `st.dataframe` como en `st.data_editor`. Los métodos de esta clase le permiten personalizar sus columnas de datos para diferentes tipos de datos, como texto, números, listas, URLs, imágenes y más. La configuración de columnas le brinda la flexibilidad de visualizar e interactuar con sus datos de la manera que prefiera. Los siguientes son los métodos de clase de `st.column_config`:

- st.column_config.Column
- st.column_config.TextColumn
- st.column_config.NumberColumn
- st.column_config.CheckboxColumn
- st.column_config.SelectboxColumn
- st.column_config.DatetimeColumn
- st.column_config.DateColumn
- st.column_config.TimeColumn
- st.column_config.ListColumn
- st.column_config.LinkColumn
- st.column_config.ImageColumn
- st.column_config.LineChartColumn
- st.column_config.BarChartColumn
- st.column_config.ProgressColumn

    Ejemplo

    ```python
    # Código column_config.py
    import streamlit as st
    import pandas as pd
    
    data_df = pd.DataFrame(
        {"widgets": ["selectbox", 
                    "st.number_input", 
                    "st.text_area", 
                    "st.button"],
        "widgets_func": ["Para seleccionar un valor", 
                         "Para seleccionar un número", 
                         "Para escribir texto",
                         "Para ejecutar una acción"],
        "widget_rank": [9, 950, 3, 1],
        "favorite": [True, False, False, True],
        "category": [
            "Data Exploration",
            "Data Visualization",
            "LLM",
            "Machine Learning"],
        })
    
    # En esta columna se valida el texto
    columna_texto = st.column_config.TextColumn("Widgets de Streamlit", 
                        help="Comandos **widget** de streamlit",
                        max_chars=50,
                        validate="^st\\.[a-z_]+$")
    
    # En esta columna se puede meter cualquier cosa
    columna = st.column_config.Column("Qué hacen",
                help="Función **widget** de streamlit",
                width="medium",
                required=True)
    
    columna_numero = st.column_config.NumberColumn("Ranking", 
                        help="Ranking del **widget**",
                        min_value=0,
                        max_value=1000,
                        step=1,
                        format="%d")
    
    columna_checkbox = st.column_config.CheckboxColumn("Tu favorito",
                            help="Elige tus favoritos",
                            default=False)
    
    opciones = ["Data Exploration", "Data Visualization", "Machine Learning"]
    columna_selectbox = st.column_config.SelectboxColumn("Categoría del widget",
                            help="Categoría del widget",
                            width="medium",
                            options=opciones)
    
    configuracion_columnas = {"widgets": columna_texto,
            "widgets_func": columna,
            "widget_rank": columna_numero,
            "favorite": columna_checkbox,
            "category": columna_selectbox}
    
    st.data_editor(data_df,
        column_config=configuracion_columnas,
        hide_index=True)
    ```
